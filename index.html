串口命令表
发送者 第一个字节 第二个字节 第三个字节 第四个字节 功能
主机 0x96 0x69 开机
0x3A 0xA3 关机
0x0F 0xF0 重启
0xF0(开启) 0x00 开启蜂鸣器
0x01 开启泵
0x02 开启阀
0x03 -0x05 开启灯
0xF1(关闭) 0x00 关闭蜂鸣器
0x01 关闭泵
0x02 关闭阀
0x03 - 0x05 关闭灯
0xF2(设置) 0x00 data 设置最大压力(mmHg)
0x01 data 设置最小压力(mmHg)
0x02 data 设置加压时间(ms)
0x03 data 设置高压时间(ms)
0x04 data 设置低压时间(ms)
0x05 data 设置蜂鸣器频率(Hz)
0x06 data 设置泵转速
0xF3(运行) 0x00 开始运行
0x01 结束运行
0x02 暂停运行
0x03 继续运行
0x04 - 运行测试程序
从机 0xA0 0x0 - 发送异常代码
0xA1 0x0 - 发送状态代码
0xA2 0x00 data 发送泵转速
0x01 data 发送气压值
0x02 data 发送温度值
0x03 data 发送电池电量
0xA3 data 发送脉搏数据
